Руководство пользователя GamePulse

1. Назначение
Поиск игр, ведение коллекции, отзывы, общение с друзьями.

2. Регистрация и вход
(См. скриншот 1) Страница регистрации: перейдите на /src/register, заполните логин (3-20 символов), email, пароль (мин. 6 символов), нажмите Зарегистрироваться.
(См. скриншот 2) Страница входа: перейдите на /src/login, введите логин и пароль, нажмите Войти.
Выход: нажмите Выход в правом верхнем углу.

3. Роли пользователей
Геймер: каталог, коллекция, отзывы, друзья, отчёты.
Администратор: всё, что может геймер, плюс управление играми, пользователями, отзывами.

4. Сценарии работы для геймера
4.1. Добавление игры в коллекцию: (См. скриншот 3) Каталог -> выбрать игру -> Подробнее -> выбрать статус и оценку -> Добавить. (См. скриншот 4) Страница игры с формой добавления.
4.2. Управление коллекцией: (См. скриншот 5) Моя коллекция -> изменить статус/оценку -> Обновить, или Удалить.
4.3. Написание отзыва: (См. скриншот 6) Страница игры -> раздел Написать отзыв -> выбрать оценку, написать текст -> Отправить.
4.4. Друзья: (См. скриншот 7) Друзья -> в блоке "Рекомендованные друзья" нажать Добавить, или найти через поиск.
4.5. Отчёты: (См. скриншот 8) Отчёты -> выбрать отчёт -> нажать Excel или Doc для выгрузки. (См. скриншот 9) Пример выгруженного отчёта в Excel. (См. скриншот 10) Пример в Word.

5. Сценарии работы для администратора
5.1. Управление играми: (См. скриншот 11) Админ-панель -> Управление играми -> Добавить игру (см. скриншот 13), редактировать или удалить (см. скриншот 12).
5.2. Управление пользователями: (См. скриншот 14) Админ-панель -> Управление пользователями -> изменить роль или удалить.
5.3. Модерация отзывов: (См. скриншот 15) Админ-панель -> Модерация отзывов -> удалить неприемлемые.

6. Часто задаваемые вопросы
Вопрос: Не могу войти. Ответ: Проверьте логин и пароль. Обратитесь к администратору.
Вопрос: Игра не добавляется в коллекцию. Ответ: Убедитесь, что вы авторизованы. Возможно, игра уже есть в коллекции.
Вопрос: Как удалить свой отзыв? Ответ: На странице игры под вашим отзывом есть кнопка Удалить.
Вопрос: Как стать администратором? Ответ: Только другой администратор может изменить вашу роль в разделе Управление пользователями.
